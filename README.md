# Features
・ユーザ登録すれば誰でも情報を投稿できるので、充実した情報量が得られる
・投票機能（ユーザ毎に上限 10 votes/day を設ける予定）によって、質の良い情報が上位に来る
・上記2つの主要な特徴のほか、タグ付けなどを活用して、「競プロ用語に関してはGoogle検索を超える検索性能を目指したい
